Overview
========
This application demonstrates how to use the GPIO module in output mode to control the RGB LED on the shield.

Hardware requirements
=====================
- FRDM-MCXA153 board
- Type-C USB cable
- Shield

Board settings
==============
- Default

Preparation
===========
- Connect the shield to the FRDM-MCXA153 board.
- Connect the type-C USB cable between the host PC and the MCU-Link port (J15) on the target board.
- Build the application.
- Run or debug the application.

Result
======
- The RGB LED blinks red, green, and blue with an interval of 500 ms.
